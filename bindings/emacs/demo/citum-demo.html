1. Introduction

This document demonstrates the oc-citum export processor, which drives the Citum citation engine over a JSON-RPC pipe — no FFI required.

1.1. Default (parenthetical) style

A standard parenthetical citation: (Chen, 2017).

Multiple references in one citation: (Chen, 2017; Webb, 1968).

A citation with a page locator: (Chen, 2020, p. 42).

1.2. Integral (narrative) style

Integral mode with [cite/t:]:

Chen (2017) argued that collaborative knowledge-making restructures reference practices.

Compare also Webb (1968) on the institutional role of footnotes.

1.3. Author-suppressed style

When only the year is needed in context — (2020) — the author name is suppressed.

1.4. Caps variant

Sentence-opening citation with [cite//c:]: (Webb, 1968) remains the canonical study of the footnote as institution.

1.5. Nocite (bibliography-only)

The following entry appears in the bibliography but not inline: (García-Reyes & Montoya, 2022).

1.6. A fuller note

Tanaka and Suzuki (2019, pp. 51–-56) analyse citation networks in Japanese-language scholarship, while (Okafor et al., 2024) provides a large-scale computational perspective.
