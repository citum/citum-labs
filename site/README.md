# Citum Website

This directory contains the source for the [citum.io](https://citum.io) website.

The site is hosted on GitHub Pages and is automatically deployed via the `.github/workflows/deploy_pages.yml` workflow.

## Local Development

You can view the site locally by opening `index.html` in any web browser, or by using a simple HTTP server:

```bash
python3 -m http.server --directory site
```
